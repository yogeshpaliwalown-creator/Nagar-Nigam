# Nagar-Nigam
A mobile application developed to help citizens report common urban issues directly to municipal authorities. The app focuses on simplifying complaint registration by allowing users to submit location-based issues with images.

#Project Overview
The Nagar Nigam Mobile Application is designed to improve communication between citizens and municipal bodies by providing a digital platform for reporting everyday civic problems. Users can report issues such as damaged 
roads, garbage accumulation, fallen electric poles, and water leakage using their mobile device.

#This project was developed as an academic and practical application with a focus on usability and real-world problem solving.

Features:-
  Report urban issues such as:
    Road pits
    Garbage problems
    Fallen electric poles
    Water leakage
  Location-based complaint submission
  Image upload support for better issue verification
  Map integration for identifying complaint locations
  Simple and user-friendly interface

Technology Used:-
  MIT App Inventor
  Google Maps / Location Services
  Image capture and upload functionality

Application Workflow:-
  User opens the application
  Selects the type of urban issue
  Captures or uploads an image
  Location is attached using map services
  Complaint is submitted for municipal review
